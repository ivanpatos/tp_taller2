Login (###### LISTO ######)

- request: 

url: /login
method: POST
query string: -
header: username, password
json: -

- respuesta: 

si salio ok, json con:
result : OK
data: username, name, mail, profilePicture, lastLocation, token

si salio mal:
result : ERROR
errorCode: codigo de error
2 - ERROR_INVALID_USERNAME
6 - ERROR_SAVING_DATA
7 - ERROR_INVALID_PASSWORD
