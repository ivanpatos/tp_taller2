1) "User": <idUser><json>

<idUser> = userName

<json>: 
- userName
- name (nombre y apellido)
- mail
- profilePicture
- lastLocation
- password
- token



2) "Folder": <idFolder><json>


si la carpeta es el directorio raíz del usuario -----> <idFolder> = <userName propietario>
si la carpeta no es el directorio raíz del usuario ----->  <idFolder> = <userName propietario + timestamp>

<json>:
- name
- deleted (boolean que indique borrado lógico)
- lastModified
- lastUser
- label (lista de etiquetas)
- owner
- users (lista de userName con los que se compartió la carpeta)
- parents -----> lista con todos los idFolder que incluyen a la carpeta

- children ----> lista de con los siguientes datos: 

Si la carpeta tiene una subcarpeta: 
type: "folder"
id: idFolder

Si la carpeta tiene un archivo:
type: "file"
id: idFile



3) "File": <idFile><json>

<id File> = <userName propietario + timestamp>

<json>:
- name
- deleted (boolean que indique borrado lógico)
- lastModified
- lastUser
- label (lista de etiquetas)
- owner
- users (lista de userName con los que se compartió la carpeta)
- parents -----> lista con todos los idFolder que incluyen al archivo

- extension
- path (path donde está ubicado el archivo en el server)
