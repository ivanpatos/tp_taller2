1) "User": <idUser><json>

<idUser> = userName

<json>: 
- userName
- name (nombre y apellido)
- mail
- profilePicture
- lastLocation
- password
- token



2) "Document": <idDocument><json>

Si el documento es la carpeta raíz del usuario -----> <idDocument> = <userName propietario>
Si el documento no es la carpeta raíz del usuario o es un archivo ----->  <idDocument> = <userName propietario + timestamp>

<json>:
- name
- deleted (boolean que indique borrado lógico)
- lastModified
- lastUser
- label (lista de etiquetas)
- owner
- users (lista de userName con los que se compartió la carpeta)
- parents
- children
- extension
- path


Si el documento es una carpeta, no va a tener extension ni path
Si el documento es un archivo, no va a tener children
